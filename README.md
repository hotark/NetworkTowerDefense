# Network Tower Defence

ネットワークをテーマにしたタワーディフェンスゲームを、AI駆動でさまざまな手法を使って実装していくプロジェクトです。

## ゲーム概要

プレイヤーはノード（タワー）を設置しエッジで接続してネットワークを構築。パケットがネットワーク上を流れ、攻撃の弾薬かつ建設のリソースとなる、ちょっと変わったタワーディフェンスです。

詳しくは [ゲーム仕様書](docs/game-design.md) を参照。

## プロジェクトの目的

同じゲームデザインを、異なるAIツール・開発手法で実装してみることで：

- AI駆動開発のワークフローを試す
- 手法ごとの向き不向きを体感する
- 最終的にUnityで本格実装するための知見を得る

## 実装バリエーション

| 実装 | 技術 | 状態 |
|------|------|------|
| モックアップ | HTML5 Canvas + vanilla JS | プロトタイプ |
| Unity版 | Unity (予定) | 未着手 |

## 開発手法

このプロジェクトでは [Kiro-style Spec Driven Development](CLAUDE.md) を採用しています。

```
要件定義 → 設計 → タスク分解 → 実装
```

各フェーズでAIと協働し、仕様から実装までを一貫して進めます。
