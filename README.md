# Network Tower Defence

ネットワークの仕組みをモチーフにしたタワーディフェンスゲーム。

パケット生成タワーから中継タワー・エッジを経由して攻撃タワーに弾薬を供給し、敵を迎撃する。
タワー間の接続（エッジ）には帯域制限があり、ネットワーク設計がゲームの鍵となる。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `mockup/` | ブラウザで動作するゲームモックアップ（HTML + Canvas + vanilla JS） |

## 開発

```bash
cd mockup
python3 -m http.server 8080
# http://localhost:8080/ で動作確認
```
