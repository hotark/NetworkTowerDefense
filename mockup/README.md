# Mockup

ビルドツールなしで動作するゲームモックアップ。HTML + Canvas + vanilla JS (ES Modules)。

## 起動

```bash
python3 -m http.server 8081
```

## ファイル構成

```
mockup/
├── index.html          # エントリポイント
├── style.css           # UI スタイル
├── main.js             # ゲームループ（固定 1/60 timestep）
├── core/
│   ├── config.js       # 定数・バランス（タワー7種/エッジ/敵5種×3Lv/30ウェーブ）
│   ├── state.js        # GameState: 全状態管理 + tick
│   ├── tower.js        # Tower エンティティ
│   ├── edge.js         # Edge エンティティ
│   ├── enemy.js        # Enemy エンティティ（5種×3レベル+ボス）
│   ├── packet.js       # Packet（エッジ上の移動物）
│   ├── wave.js         # WaveManager: 30ウェーブ管理
│   └── map.js          # ステージデータ（S字パス/スポット/拠点）
├── render/
│   └── renderer.js     # Canvas 描画
├── input/
│   └── inputHandler.js # 入力 + UI パネル管理
└── assets/
    ├── towers/         # タワー SVG（7種 + 拠点）
    └── enemies/        # 敵 SVG（5種）
```

## ゲーム概要

### タワー（7種）

**生成タワー** — パケットを生成してエッジに送信
- ユニキャスト: 大量パケットを順次送信（1エッジずつラウンドロビン）
- ブロードキャスト: 全出力エッジに同時送信

**中継タワー** — パケットを中継・増幅
- 増幅（リピーター）: パケットの charge を倍率で増幅して転送
- 分配（ディストリビューター）: charge を平坦化して全出力エッジに均等分配

**攻撃タワー** — パケットを弾薬に変換して敵を攻撃
- 速射: 高速・低火力
- 重撃: 低速・高火力・狭射程
- 狙撃: 中速・中火力・広射程

### 敵（5種 × 3レベル）

- 通常: 標準的な敵
- 高速: スピード速い・低HP
- 重装: 高HP・低速
- 破壊工作: タワー/エッジを攻撃・遅い
- 急襲: タワー/エッジを攻撃・速い

### エッジ

タワー間を接続する通信路。帯域制限あり。
使用率に応じて色が変化（青→黄→赤）。
